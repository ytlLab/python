### 關鍵字參數：參數名稱=值

所謂的關鍵字參數(keyword arguments)是指呼叫函數時，參數是用參數名稱=值配對方式呈現。

```
#練習

def interest(interest_type, subject):
	""" 顯示興趣和主題 """
	print("我的興趣是 " + interest_type )
	print("在 " + interest_type + " 中, 最喜歡的是 " + subject)
	print( )
interest(interest_type = '旅遊', subject = '敦煌')  # 位置正確
interest(subject = '敦煌', interest_type = '旅遊')  # 位置更動
```
參數預設值的處理，特別需留意：函數設計時含有預設值的參數，必須放置在參數列的最右邊
```
#練習

def interest(interest_type, subject = '敦煌'):
	""" 顯示興趣和主題 """
	print("我的興趣是 " + interest_type )
	print("在 " + interest_type + " 中, 最喜歡的是 " + subject)
	print( )
interest('旅遊')                                 	# 傳遞一個參數
interest(interest_type = '旅遊')                 	# 傳遞一個參數
interest('旅遊', '張家界')                       	# 傳遞二個參數
interest(interest_type = '旅遊', subject = '張家界') # 傳遞二個參數
interest(subject = '張家界', interest_type = '旅遊') # 傳遞二個參數
interest('閱讀', '旅遊類')            	# 傳遞二個參數,不同的主題
```

## 轉換英文字母大小寫

這些方法只有轉換，並沒有改變原本的字串。
- upper() 將所有字母改為大寫
- lower() 將所有字母改為小寫
- capitalize() 將字串中的第一個字母大寫，其餘小寫
- title() 將每個詞第一個字母大寫，其餘小寫

```
#練習

word = 'how are you?'
print(word.upper())  	 
print(word.lower())   	 
print(word.capitalize())   
print(word.title())    	 
```

## split() 拆出多個字串
split() 預設以空白字元來切割，並回傳包含分拆後各個字串的串列（list），回傳的資料型態是串列。

```
#練習

print('You work very hard!'.split())
#['You', 'work', 'very', 'hard!']
#可以在括號那指定分割的依據，這個分隔字串不會留下。
print('You work very hard!'.split('o'))
#['Y', 'u w', 'rk very hard!']
```

## 使用 join() 將字串結合
這個方法會需要字串（string） 與串列（list）這兩個材料，利用字串當膠水，把串列中的各個字串結合起來，回傳的值是串列（list）。
```
#練習

gift_list = ['iphone 12', 'PS5', 'notebook']
print('Christmas gifts: '+', '.join(gift_list))
```

## replace() 替換

原本的字串值並不會改變，只是加上replace()方法後，指定的部分會被替換，有點類似拿了一個副本做替換，原本的字串不會改變。
```
#練習

lesson ='I am learning this year : Python.'
print(lesson.replace('Python', 'playing guitar'))
#I am learning this year : playing guitar.
print(lesson)
#I am learning this year : Python.
```

## index() 或 find() 找尋位置
```
#練習

lesson ='the new skill I am learning this year : Python.'
print(lesson.index('the'))
#0
print(lesson.find('the'))
#0
print(lesson.index('I'))
#14
print(lesson.find('I'))
#14
```
如果想從右邊開始找可以在 find() 與 index() 前面加上 r，變成rfind()、rindex()，

就可以從右邊開始找
以找尋 e 這個字母為例，會得到不一樣的位置回覆。
```
#練習

print(lesson.find('e'))
#2
print(lesson.rfind('e'))
#34
print(lesson.index('e'))
#2
print(lesson.rindex('e'))
#34
```

當找不到目標時，index() 方法會回覆 ValueError，find() 則回傳 -1
```
#練習

lesson ='the new skill I am learning this year : Python.'
print(lesson.find('peace'))
#-1
print(lesson.index('peace'))
#Traceback (most recent call last):
#  	print(lesson.index('peace'))
#ValueError: substring not found
```
