# 布林值
表達True或False(是與否)</br>
布林值是電腦科學裡面一個重要的資料型態，由發明布林代數的數學家命名。在程式語言裡面，有許多是與否的判斷，會用在判斷式、迴圈、回傳值之中，是電腦做決策來源指令之一。</br>
在使用上，布林值十分簡單，就只有兩個原始類型，分別是True和False，用來區別是與否。</br>
```
#練習布林值
#布林值 (是或否)
c=1+1==3 #判斷1+1是否等於3，回傳答案
print(c) #印出False
```
# 資料型別轉換
Python具有簡單的資料型別自動轉換，若是整數和浮點數運算，會把整數轉成浮點數再運算，結果會顯示出浮點數。</br>
num1=5+7.8 #結果為12.8，浮點數。</br>
若是數值與布林值運算，會把布林值轉成數值再運算</br>
num2=5+True #結果為6，整數。</br>

## 資料型別轉換函數
int()：強制轉換為整數</br>
float()：強制轉換為浮點數</br>
str()：強制轉換為字串</br>

```
#練習資料型別轉換函數
#num=23+"67"
num=23+int("67")
print(num)
score=60
#print("小明的成績為"+score)
print("小明的成績為"+str(score))
```

# 參數格式化
print()支援參數格式化功能 %s代表字串、%d代表整數、%f代表浮點數
```
name="林小明"
socre=80
print("%s的分數為%d" %(name,socre))
```
# 控制列印位置
%5d：列印5個字元，少於5的會在數字左方填入空白，大於則會全部列印</br>
%5s：列印5個字元，少於5的會在字串左方填入空白，大於則會全部列印</br>
%8.2f：固定列印8個字元(含小數點)，小數固定列印2位數，</br>
小數少於2位數會填入0字元</br>
```
price=23.8
print("價格為%8.2f" %price) #印出 ___23.80 (底線為三個空白字元)
```
